## goの記法構成

```go

package main // 各Goファイルのメイン関数をこの場所で定義する

import "fmt" // ここでパッケージのインポートを行う。goはパッケージは細かく分かれているため、ここにたくさん記載が増える。使用していないパッケージを使用していると怒られる。各パッケージの使用方法は都度覚えればいいと思う。

// ここが関数。apiなどを作成して、複数の処理が分裂する場合は、ここは各関数を起動させるハブのようになり、細かい処理は中の各関数で処理される。
func main() {
    ask()
}

func ask() {
    name := "Big〇〇ジョン"
    age int = 21
    fmt.printf("僕の名前は、「%sです。\n%sです",name,age) // 1つ目の%sのところにname、2つ目にageが入る。
}

```
