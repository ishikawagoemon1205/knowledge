## コマンドラインの使い方

```markdown

---作成系---
touch <新しく作成するファイル名>　：新しいファイルを作成する方法
mkdir <新しいフォルダ名>　：新しいフォルダを作成する方法

---参照系---
cat <確認したいファイル名>　：ファイルの中に下記が記載されているか確認だけ出来る機能
pwd ：カレントディレクトリ参照する方法
ls　：現在のディレクトの配下にあるファイル、フォルダを一覧で表示する方法

---移動系---
cd <移動したいフォルダ名> ：移動したいフォルダにディレクトリが移動する
cd .. ：一つ上の階層のディレクトリに移動する
cd　：ホームディレクトリに移動する
cd /.　：OSの一番上のディレクトリに移動する
mv <移動させたファイル名、ディレクトリ名> <移動先のディレクトリ名>　：ファイルの移動

---名称変更---
mv <現在のファイル名> <新しいファイル名>　：ファイルの名称変更
mv <現在のディレクトリ名> <新しいディレクトリ名>　：ディレクトリ名称変更

---複製---
cp -r <コピーするファイル名> <新しいファイル名>　：ファイルのコピー
cp -r <コピーするディレクトリ名> <新しいディレクトリ名>　：ディレクトリのコピー

---削除---
rm <削除するファイル名>　：ファイルの削除
rm　-r <削除するディレクトリ名>　：ディレクトリの削除

```

## ファイルを編集する方法

```markdown

vim <ファイル名>　：ファイルをvimエディタで編集
    - i　：編集モード起動
    - Esc → :wq　→ Enter　：保存して終了
    - Esc → :ql　→ Enter　：保存せずに終了

```
